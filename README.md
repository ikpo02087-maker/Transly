# Transly - Speak & Translate Web App

**Transly** is a web application that allows users to **convert speech to text** and **translate it into multiple languages** in real-time.  

This app is designed to be **fast, simple, and mobile-friendly**, making it easy to speak and get translations instantly.

---

## **Features**
- Real-time speech recognition using Web Speech API  
- Instant translation into multiple languages  
- Works on desktop and mobile browsers  
- Clean, responsive interface  

---

## **Technologies Used**
- HTML  
- CSS  
- JavaScript  
- Web Speech API  
- Optional: Translation API (e.g., Google Translate API)  

---

## **How to Run**
1. Open `index.html` in a browser  
2. Allow microphone access  
3. Speak and get instant translations
